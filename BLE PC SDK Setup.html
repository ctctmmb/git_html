PC SDK Setup

Embitz Install:
  1. Download IDE package from folder ,Go to Embitz directory and click install EmBitz_1.0.0.exe



Apply Patch: 
  1. From folder go to EmBitzIDE PATCH_0414\patch\lib and copy cmsis folder



  2. Copy patch folder "cmsis" to EmBitz installation path "C:\Program Files (x86)\EmBitz\1.00\share\em_armgcc\lib"
    (Lib path may different depending to your OS)



  3. Go to patch folder "EmBitzIDE PATCH_0414\patch" and copy folder "nordic"



  4. Go to Install path of EmBitz in "C:\Program Files (x86)\EmBitz\1.00\share\EmBitz\templates\wizard\arm"
    Delete "nordic" folder 



  5. Copy the patch "nordic" folder to Install path of EmBitz at
    "C:\Program Files (x86)\EmBitz\1.00\share\EmBitz\templates\wizard\arm"



Project Import: 
  1. Run "EmBitz" and open a new nordic project 
    File -> New -> Project, Select Nordic-ARM





  2. AKM demokit use MTM904 which is Nordic based nRF51822 Rev3 (xxAC) Integrated with Antenna. 
    Type Project title, and select  workspaces location, click Next



  3. Select "ARM GCC Compiler", click Next


  4. Select "nRF51822 (Cortex M0)", click Next  
    Select "nRF51822_AC", click Next
      

  5. Select "S130v1 Soft Device Integration"  , click checkbox "Create Hex File", click button "Finish".


  6. Check if Setting for the debugger are same as below picture
      

  7. Go to "Settings -> Tools -> Global compiler settings -> Build Options" ,
    Click checkbox:
    Explicitly add currently compiling file’s directory to compiler search dir
    Explicitly add project’s top-level directory to compiler search dirs





  8. Your Project is ready to be compiled. Right click on Project file and select Build. For attempt build





SDK Structure Explannation: 

Sources : Collection of all .C & .CPP files.
Arduino SDK : All Library related to Arduino Function Internal
Library : Ported External Library (Gazell, SPI, Watchdog, Wire)
Mbed SDK : All library related to MED2.0 (Mbed Driver)
AKxxxx.cpp : All AKM sensor function Implementation
I2C.cpp : I2C helper function
Main.cpp : Main function
S130v1.c : Nordic Soft Device S130 binary c files
Header : Collection of all header files
Nordic SDK : All function definition related to Soft Device
ASM Sources : nrf51 assembly startup codes
Others :
Bunch of examples related to AKM sensor usage
Linker script
